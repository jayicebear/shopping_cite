
# 🛒 상품 리스트 쇼핑몰 (Product Shopping List)

전자, 의류, 식품 등 다양한 카테고리의 상품을 한눈에 보고,  
검색·카테고리 필터·정렬로 원하는 상품을 빠르게 찾을 수 있는 반응형 쇼핑 사이트예요!

## ✨ 주요 기능

- 상품 카드 형식으로 깔끔한 리스트 표시 (이미지 + 이름 + 카테고리 + 가격)
- 상단 검색바로 상품명 실시간 검색
- 카테고리 필터 (전체 / 전자 / 의류 / 식품 등)
- 정렬 기능 (가격 낮은 순 / 높은 순 / 이름 순 등)
- 반응형 디자인 (모바일, 태블릿, 데스크톱 모두 예쁨)
- 상태 관리로 부드러운 UI 업데이트

## 🛠️ 기술 스택

- **Frontend**: React (Vite 또는 Create React App 추천)
- **상태 관리**: React `useState` + `useEffect`
- **스타일**: CSS 
- **배포**: Vercel, Netlify 등 정적 호스팅 추천

## 🚀 시작하기

```bash
# 저장소 클론
git clone https://github.com/jayicebear/react-shopping-list.git
cd react-shopping-list

# 의존성 설치
npm install

# 개발 서버 실행
npm start
